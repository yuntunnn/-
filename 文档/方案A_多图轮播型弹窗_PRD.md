## 一、需求背景

### 1.1 业务目标

通过PC端锁屏广告位弹窗推广夸克浏览器，以**痛点教育 + 功能展示**的方式提升用户认知，引导目标用户下载/体验夸克浏览器。

### 1.2 目标用户

| 用户画像 | 核心痛点 | 推广切入点 |
|---------|---------|-----------|
| 职场人 | 办公效率低、PDF处理难 | AI搜索 + 超级PDF |
| 影音爱好者 | 存储空间不足、下载慢 | 6T网盘 + 极速下载 |
| 被广告骚扰的人 | 浏览体验差 | 无广告浏览 |

### 1.3 核心策略

> **痛点轮播 → 功能认知 → 行动转化**

通过3帧轮播图分别展示用户核心痛点及夸克的解决方案，建立产品认知后引导用户点击CTA按钮。

## 二、功能需求

### 2.1 弹窗整体结构

```
┌──────────────────────────────┐
│                           ✕  │  ← 关闭按钮
│  ┌──────────────────────────┐│
│  │                          ││
│  │   📸 轮播图 + 文案叠加    ││  ← 轮播区域（3帧）
│  │   ← 图1 | 图2 | 图3 →   ││
│  │   ▓▓▓▓▓▓▓▓▓▓ (渐变遮罩)  ││
│  │   主标题（白色大字）       ││
│  │   副标题（白色小字）       ││
│  └──────────────────────────┘│
│  ━━━━━━━ (进度条) ━━━━━━━━━━ │  ← 轮播进度条
│         ● ○ ○  (指示器)      │  ← 圆点指示器
│                              │
│  夸克×千问 · 更强大的AI浏览器  │  ← 品牌标识（固定）
│                              │
│  ┌──────────────────────────┐│
│  │      🔵 立即体验          ││  ← CTA按钮
│  └──────────────────────────┘│
│         以后再说              │  ← 跳过文案
└──────────────────────────────┘
```

### 2.2 模块详细说明

#### 模块①：轮播图区域

| 属性 | 规格 |
|------|------|

**轮播内容（3帧）：**

| 帧 | 图片内容 | 图内主标题 | 图内副标题 |

**文案叠加样式：**

## 三、交互规格

### 3.1 弹窗生命周期

| 阶段 | 行为 |
|------|------|
| **弹出** | 遮罩层淡入 0.3s + 弹窗从底部弹入 0.4s（弹性缓动） |
| **展示** | 轮播自动播放，进度条跟随 |
| **关闭** | 淡出 0.25s 后隐藏。触发条件：✕按钮 / 以后再说 / 点击遮罩 |

### 3.2 轮播交互

| 操作 | 行为 |
|------|------|
| 自动轮播 | 3秒间隔循环，到末帧后回到首帧 |
| 手动滑动 | 触摸/鼠标拖动超过40px触发切帧 |
| 左右箭头 | hover轮播区域时显示，点击切帧 |
| 圆点指示器 | 点击跳转到对应帧 |
| 鼠标悬停 | 暂停自动轮播，移出恢复 |
| 触摸操作 | 触摸开始暂停轮播，触摸结束恢复 |

### 3.3 进度条逻辑

- 每帧开始时重置为0%
- 以30ms为间隔线性增长至100%
- 切帧时无论手动还是自动，均重置进度条

## 五、数据埋点

### 5.1 曝光事件

| 事件名 | 触发时机 | 参数 |
|--------|---------|------|
| `popup_show` | 弹窗展示 | popup_type=carousel, slide_count=3 |
| `slide_view` | 轮播帧曝光 | slide_index, slide_title, view_type=auto/manual |

### 5.2 交互事件

| 事件名 | 触发时机 | 参数 |
|--------|---------|------|
| `cta_click` | 点击CTA按钮 | popup_type=carousel, current_slide |
| `popup_close` | 关闭弹窗 | close_method=x_btn/skip/overlay, viewed_slides, duration_ms |
| `slide_switch` | 手动切帧 | from_slide, to_slide, switch_method=swipe/arrow/dot |

### 5.3 核心指标

| 指标 | 定义 | 目标 |
|------|------|------|
| 弹窗曝光量 | popup_show事件次数 | — |
| CTA点击率 | cta_click / popup_show | ≥ 5% |
| 关闭率 | popup_close / popup_show | ≤ 70% |
| 平均浏览帧数 | viewed_slides均值 | ≥ 2 |
| 平均停留时长 | duration_ms均值 | ≥ 5s |

## 六、异常处理

| 场景 | 处理方式 |
|------|---------|
| 轮播图加载失败 | 显示纯色背景 + 文案叠加层，不影响轮播功能 |
| CTA跳转链接失败 | 新标签页打开，若链接不可用做兜底跳转至官网 |
| 弹窗被浏览器拦截 | 降级为页面内嵌提示条 |
