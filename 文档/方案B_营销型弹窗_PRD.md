## 一、需求背景

### 1.1 业务目标

通过PC端锁屏广告位弹窗推广夸克会员服务，以**限时优惠 + 紧迫感营造**的方式直接驱动用户付费转化，提升会员开通率和营收。

### 1.2 目标用户

| 用户画像 | 核心需求 | 推广切入点 |
|---------|---------|-----------|
| 网盘重度用户 | 大容量存储、极速下载 | 网盘SVIP 1.9折（4.8元/月） |
| AI工具爱好者 | 深度思考、AI翻译 | AI会员 1.9折（4.8元/月） |
| 文档扫描需求者 | 证件扫描、OCR、PDF转换 | 扫描王会员 1元/7天试用 |
| 综合用户 | 多功能需求 | 夸克会员 5折年卡 |

### 1.3 核心策略

> **限时优惠 → 紧迫感 → 社会认同 → 立即行动**

通过大幅折扣吸引注意、倒计时制造紧迫感、已开通人数强化社会认同，三重驱动用户点击CTA完成转化。

## 四、交互规格

### 4.1 弹窗生命周期

| 阶段 | 行为 |
|------|------|
| **弹出** | 遮罩层淡入 0.3s + 弹窗底部弹入 0.45s（弹性缓动） |
| **展示** | 倒计时自动运行，社会认同数字自增 |
| **关闭** | 淡出 0.25s。触发条件：✕按钮 / 不再提醒 / 点击遮罩 |

### 4.2 倒计时逻辑

- 从 23:59:59 开始，每秒递减
- 到 00:00:00 后停止，不循环
- 数字变化时触发翻牌动画

### 4.3 CTA 跳转

|------|---------|
| B-1 | 网盘SVIP充值页 |
| B-2 | AI会员充值页 |
| B-3 | 扫描王试用页 |
| B-4 | 夸克会员综合充值页 |

## 六、数据埋点

### 6.1 曝光事件

| 事件名 | 触发时机 | 参数 |
|--------|---------|------|
| `popup_show` | 弹窗展示 | popup_type=marketing, variant=B-1/B-2/B-3/B-4 |

### 6.2 交互事件

| 事件名 | 触发时机 | 参数 |
|--------|---------|------|
| `cta_click` | 点击CTA | variant, cta_text, countdown_remaining |
| `popup_close` | 关闭弹窗 | close_method=x_btn/dismiss/overlay, duration_ms |

### 6.3 核心指标

| 指标 | 定义 | 目标 |
|------|------|------|
| CTA点击率 | cta_click / popup_show | ≥ 8% |
| 关闭率 | popup_close / popup_show | ≤ 65% |
| 付费转化率 | 付费成功 / cta_click | ≥ 15% |
| 平均停留时长 | duration_ms均值 | ≥ 4s |

## 七、异常处理

| 场景 | 处理方式 |
|------|---------|
| 充值页面不可用 | 新标签页兜底跳转至官网 |
| 倒计时结束 | 保持显示 00:00:00，CTA按钮依然可点击 |
| 背景动画不支持 | 降级为纯色背景，不影响核心功能 |
