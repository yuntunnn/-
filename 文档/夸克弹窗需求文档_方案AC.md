### 2.2 各模块说明

#### ① 右上角操作区

| 元素 | 说明 |
|------|------|
| 关闭按钮 ✕ | 右上角，点击关闭弹窗 |
| 「以后再说」| 关闭按钮左侧，点击关闭弹窗 |

#### ② 轮播图区域

3帧轮播，每帧为一张功能亮点图片 + 底部文案叠加层（渐变遮罩）。

| 帧 | 图片内容 | 标题 | 副标题 |
|----|---------|------|-------|
| 1 | AI搜索场景 | 搜一次，就够了 | AI深度思考，一次给你完整答案 |
| 2 | PDF编辑场景 | 扫描件秒变可编辑 | 字体颜色完美还原，PDF也能轻松改 |
| 3 | 无广告浏览场景 | 浏览无广告 | 纯净浏览，拒绝干扰 |

**交互行为：**
- 自动轮播，3秒切换一次
- 支持手动左右滑动切换
- hover显示左右箭头按钮
- 鼠标悬停暂停自动轮播
- 底部进度条跟随轮播进度填充
- 圆点指示器可点击跳转

#### ③ 品牌主标题

- 文案：**夸克×千问 · 更强大的AI浏览器**
- 「夸克×千问」用蓝紫渐变色高亮
- 固定展示，不随轮播变化

#### ④ CTA按钮

| 属性 | 说明 |
|------|------|
| 文案 | 立即体验（固定） |
| 点击行为 | 跳转夸克下载/体验页 |
| 效果 | hover上浮 + 阴影加深 |

### 2.3 弹窗样式要求

| 属性 | 规格 |
|------|------|
| 弹窗宽度 | 340px |
| 弹窗圆角 | **无圆角** |
| 按钮圆角 | **无圆角** |

## 四、通用交互规格

### 4.1 弹窗弹出/关闭

| 动作 | 效果 |
|------|------|
| 弹出 | 遮罩层淡入 + 弹窗从底部弹入（带弹性缓动） |
| 关闭 | 整体淡出。触发方式：✕按钮 / 以后再说·不再提醒 / 点击遮罩空白区域 |

### 4.2 关闭后行为

| 操作 | 后续行为 |
|------|---------|
| 点击 ✕ | 关闭弹窗，下次仍可展示 |
| 点击「以后再说/不再提醒」| 关闭弹窗，后续不再展示 |
| 点击遮罩空白区域 | 关闭弹窗，下次仍可展示 |

## 五、弹窗埋点

### 5.1 埋点事件列表

| 序号 | 事件名 | 触发时机 | 说明 |
|------|--------|---------|------|
| 1 | `popup_show` | 弹窗展现 | 弹窗成功渲染并展示给用户时上报 |
| 2 | `popup_btn_click` | 点击CTA按钮 | 用户点击「立即体验」或「立即开通」 |
| 3 | `popup_close` | 关闭弹窗 | 通过 ✕ 按钮关闭 |
| 4 | `popup_dismiss` | 不再提示 | 通过「以后再说」或「不再提醒」关闭 |
| 5 | `popup_avoid` | 规避 | 用户点击遮罩空白区域关闭弹窗 |
| 6 | `install_start` | 开始安装 | 用户点击CTA后开始下载/安装夸克 |
| 7 | `install_finish` | 安装结束 | 夸克安装完成 |

### 5.2 公共参数

每个事件均需携带以下参数：

| 参数 | 说明 | 示例 |
|------|------|------|
| `popup_type` | 弹窗方案类型 | `carousel`（方案A）/ `hybrid`（方案C） |
| `popup_version` | 弹窗版本 | `v1.0` |
| `timestamp` | 事件时间戳 | `1709020000` |
| `user_id` | 用户标识 | — |
| `device_id` | 设备标识 | — |

### 5.3 各事件专属参数

| 事件 | 专属参数 | 说明 |
|------|---------|------|
| `popup_show` | `current_slide` | 展现时当前轮播帧索引 |
| `popup_btn_click` | `current_slide`, `btn_text` | 点击时当前帧 + 按钮文案 |
| `popup_close` | `duration_ms`, `viewed_slides` | 停留时长 + 已浏览帧数 |
| `popup_dismiss` | `duration_ms`, `viewed_slides` | 同上 |
| `popup_avoid` | `duration_ms`, `viewed_slides` | 同上 |
| `install_start` | `source_slide` | 触发安装时所在帧 |
| `install_finish` | `install_duration_ms` | 安装耗时 |

### 5.4 核心指标

| 指标 | 计算方式 |
|------|---------|
| 曝光量 | `popup_show` 总次数 |
| 按钮点击率 | `popup_btn_click` / `popup_show` |
| 关闭率 | (`popup_close` + `popup_dismiss` + `popup_avoid`) / `popup_show` |
| 不再提示率 | `popup_dismiss` / `popup_show` |
| 安装转化率 | `install_start` / `popup_btn_click` |
| 安装完成率 | `install_finish` / `install_start` |
| 整体转化率 | `install_finish` / `popup_show` |
