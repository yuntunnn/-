## 一、需求背景

### 1.1 业务目标

融合方案A（痛点轮播教育）和方案B（限时营销转化）的优势，在单个弹窗中展示**多种会员产品**，通过轮播实现多产品曝光，同时保留营销转化元素驱动付费。

### 1.2 方案定位

| 对比 | 方案A 轮播型 | 方案B 营销型 | **方案C 融合型** |
|------|------------|------------|----------------|
| 核心目标 | 品牌认知 | 付费转化 | **认知 + 转化** |
| 产品展示 | 功能亮点 | 单个会员 | **多个会员轮播** |
| 营销元素 | 无 | 倒计时+社会认同 | **倒计时** |
| 适用场景 | 新用户首次触达 | 精准会员推广 | **全场景通用** |

### 1.3 核心策略

> **多产品轮播曝光 → 品牌认知建立 → 限时优惠驱动 → 立即转化**

通过3帧轮播展示不同会员产品及优惠信息，以产品名+折扣大字为核心视觉，CTA按钮固定文案「立即开通」，用户无论停留在哪一帧都能直接转化。

### 2.2 模块详细说明

#### 模块①：关闭与跳过（右上角）

| 元素 | 规格 |
|------|------|
| 关闭按钮 | 右上角 28×28px，白色半透明圆形，`top:12px; right:12px` |

#### 模块②：轮播图区域

| 属性 | 规格 |
|------|------|
| 尺寸 | 宽度100%，高度 220px |

## 三、交互规格

### 3.1 轮播交互

| 操作 | 行为 |
|------|------|
| 自动轮播 | 3秒间隔循环播放 |
| 手动滑动 | 拖动超过40px触发切帧 |
| 左右箭头 | hover轮播区域时显示 |
| 圆点指示器 | 点击跳转 |
| 鼠标悬停 | 暂停自动轮播 |
| 触摸操作 | 触摸时暂停，释放后恢复 |

### 3.2 动态内容同步

轮播切帧时，以下元素**同步更新**：

| 元素 | 更新内容 |
|------|---------| 
| 限时福利条 | 促销标签文案切换（「新人限时优惠」↔「新人限时试用」） |

以下元素**不随轮播变化**：

| 元素 | 说明 |
|------|------|
| 品牌标识主标题 | 始终显示「夸克×千问 · 更强大的AI浏览器」 |
| CTA按钮 | 始终显示「立即开通」 |
| 飘带标签 | 始终显示「限时福利」 |
| 倒计时 | 持续运行，不因切帧重置 |

### 3.3 弹窗生命周期

| 阶段 | 行为 |
|------|------|
| **弹出** | 遮罩淡入 0.3s + 弹窗底部弹入 0.45s（弹性缓动） |
| **展示** | 轮播自动播放 + 倒计时运行 |
| **关闭** | 淡出 0.25s。触发：✕ / 不再提醒 / 点击遮罩 |

## 五、数据埋点

### 5.1 曝光事件

| 事件名 | 触发时机 | 参数 |
|--------|---------|------|
| `popup_show` | 弹窗展示 | popup_type=hybrid |
| `slide_view` | 轮播帧曝光 | slide_index, member_type, view_type=auto/manual |

### 5.2 交互事件

| 事件名 | 触发时机 | 参数 |
|--------|---------|------|
| `cta_click` | 点击CTA | current_slide, countdown_remaining |
| `popup_close` | 关闭弹窗 | close_method=x_btn/dismiss/overlay, viewed_slides, duration_ms |
| `slide_switch` | 手动切帧 | from_slide, to_slide, switch_method |

### 5.3 核心指标

| 指标 | 定义 | 目标 |
|------|------|------|
| CTA点击率 | cta_click / popup_show | ≥ 6% |
| 各帧CTA分布 | 各帧cta_click占比 | 监控各会员产品转化差异 |
| 平均浏览帧数 | viewed_slides均值 | ≥ 2 |
| 付费转化率 | 付费成功 / cta_click | ≥ 12% |
| 关闭率 | popup_close / popup_show | ≤ 65% |

## 六、方案C vs 方案A/B 的核心差异

| 维度 | 方案A | 方案B | **方案C** |
|------|------|------|---------| 
| 产品曝光 | 功能展示，不涉及会员 | 单一会员产品 | **3种会员产品轮播** |
| 转化路径 | 下载/体验 | 付费开通 | **付费开通（多产品可选）** |
| CTA | 固定 | 固定 | **固定「立即开通」** |
| 营销元素 | 无 | 倒计时+社会认同 | **倒计时** |
| 品牌标识 | 固定标题 | 固定标题 | **固定主标题（20px加粗）** |
| 视觉核心 | 轮播图片 | 折扣大字+功能图标 | **产品名+折扣大字** |
| 用户决策 | 「要不要试」 | 「要不要买」 | **「买哪个」** |

## 七、异常处理

| 场景 | 处理方式 |
|------|---------|
| 充值页面不可用 | 新标签页兜底跳转至官网 |
| 倒计时结束 | 保持显示 00:00:00，CTA按钮依然可点击 |
| 背景动画不支持 | 降级为纯色背景，不影响核心功能 |
